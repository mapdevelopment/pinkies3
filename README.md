# Pinkies Robot

This repository contains the firmware and code for the Pinkies autonomous robot.

## 🚀 Getting Started

### Installation & Usage
Clone the repo:
```bash
git clone https://github.com/mapdevelopment/pinkies3.git
```

**Open in VS Code:** 
Open the cloned folder directly in your editor.

**Build & Upload:**
1. Use the PlatformIO icon in the left sidebar.
2. Click Build to compile the code.
3. Click Upload to flash the firmware to your board.

## 📂 Folder Structure
- ```/src```: Main source code.

- ```/include```: Header files.

- ```/lib```: Internal libraries used by the project.

- ```/python```: Helper scripts for Raspberry Pi Zero 2W

- ```/platformio.ini```: Project configuration and settings.
